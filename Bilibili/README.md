## 简介及说明

1. 常用页面夜间模式全覆盖
2. 去除广告
3. 页面样式优化

```ruby
㊖ 凡涉及到金钱交易（充值、付费购买等）等页面，若遇到因本样式导致文字颜色看不清的情况，请关闭本样式之后再进行金钱交易操作。
否则因未关闭本样式导致的财产损失与本样式及作者无关。

㊖ 部分充值及支付相关元素不进行夜化。

若发现文字看不清的情况请及时反馈。
```
